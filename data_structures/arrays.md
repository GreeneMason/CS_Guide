# Arrays

A data structure that uses indexing starting at 0. Each index(input) has an element(output). Very commonly used for large ammounts of data. 

## For Java
- Declare array

'int[] g;'

- Create array

'g = new int[10];'

-Initialize or populate array

'for(int i = 0; i < n; i++){
a[i] = i;
}'
