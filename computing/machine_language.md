### Definitions
Function: A flow of control contruct.

Standard Output: A set language, syntax, or encoding so the results are accessible.

Standard Input: A set language or syntax in which the instructions/data are accessible to the computer.

Booting: Using a small bit of memory to quickly access the rest of memory.

Dumping: Removing data or instructions.

### Programs that proccess programs
 - Assembler- Assembly language allows for the use of symbollic names for operations and machine addresses.
 - Interpreter- A program that excecutes the code/instructions given to it.
 - Compiler- A program that takes code and turns it into machine code so it can excecute the program.
 - Virtual Machine- A machine that can run programs and excecute code but it has no hardware.
